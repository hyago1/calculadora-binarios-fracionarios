## Calculadora de binario fracionario.

### Funções.
* Calcula numeros binarios fracionarios.
* Mostra valores ieee754 com precisão simples.
* Ensina o Passo a passo da conversão para ieee754.

### Tecnologias usadas:
* HTML.
* CSS.
* JavaScript.
